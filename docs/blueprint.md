# **App Name**: HabitZen

## Core Features:

- Add Habits: Add new habits with a title and optional description.
- Mark Completion: Mark habits as complete for the current day with a simple toggle.
- Calendar Log: View a calendar log of habit completion, showing which habits were completed on which days.
- Daily Reminders: Set daily reminders for each habit with configurable times. Reminders persist locally.
- AI Habit Suggestions: Use AI tool to suggest related, new habits based on existing habits, and/or the user's goals. AI will reason whether/when a suggestion makes sense to the user.

## Style Guidelines:

- Primary color: Forest green (#34A04A) for growth and balance.
- Background color: Off-white (#F7F9F7) to ensure a clean and minimalist look.
- Accent color: Light green (#A7D1AB) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look; suitable for both headlines and body text.
- Use simple, line-based icons for habits and navigation.
- Maintain a clean, uncluttered layout with ample whitespace to emphasize simplicity.
- Use subtle animations to transition between views and to provide feedback on user interactions.
